# Покрокова інструкція: деплой гри на Vercel

Робіть **саме в цьому порядку**. Якщо проєкт на Vercel у вас уже є — краще його видалити і створити новий за цими кроками.

---

## Крок 1. Відкрити створення проєкту

1. Зайдіть на **https://vercel.com** і увійдіть у свій акаунт.
2. Натисніть кнопку **«Add New…»** (або **«New Project»**).
3. Оберіть **«Import Git Repository»** і знайдіть репо **BigPersik/forest-cluster** (або як у вас названий репо з грою).
4. Натисніть **«Import»** біля цього репо.

---

## Крок 2. Root Directory = `client`

На екрані налаштувань проєкту:

1. Знайдіть блок **«Root Directory»**.
2. Натисніть **«Edit»** (або «Змінити»).
3. У полі введіть **саме:** `client`
4. Підтвердіть (натисніть **Continue** або галочку).

Це обов’язково: Vercel має збирати саме папку **client**, а не корінь репо.

---

## Крок 3. Виправити Output Directory (щоб не було «public»)

1. Трохи нижче знайдіть блок **«Build and Output Settings»** (або «Налаштування збірки та виводу»).
2. Розгорніть його (натисніть на блок або «Override»).
3. Знайдіть поле **«Output Directory»** (Вихідний каталог).
4. Якщо там написано **`public`** — **видаліть** і впишіть: **`dist`**
5. Якщо поле порожнє — впишіть: **`dist`**

Без цього кроку Vercel шукає папку `public` і показує помилку. Наш проєкт збирається в папку **`dist`**.

---

## Крок 4. Змінна середовища для RGS

1. Знайдіть блок **«Environment Variables»** (Змінні середовища).
2. У полі **Name** введіть: `VITE_RGS_URL`
3. У полі **Value** введіть URL вашого RGS на Render, наприклад:  
   `https://forest-cluster.onrender.com`  
   (без слеша в кінці, свій URL скопіюйте з Render.)
4. Натисніть **Add** (Додати).

---

## Крок 5. Запустити деплой

1. Переконайтесь, що:
   - **Root Directory:** `client`
   - **Output Directory:** `dist`
   - **Environment Variable:** `VITE_RGS_URL` = URL з Render
2. Натисніть **«Deploy»** (Розгорнути).
3. Дочекайтесь завершення збірки (зазвичай 1–2 хвилини).

Якщо все зроблено правильно, статус стане **«Ready»** і з’явиться посилання на сайт (на кшталт `https://forest-cluster-xxx.vercel.app`). Це посилання можна надсилати друзям для гри.

---

## Якщо проєкт уже існує і ви хочете лише виправити налаштування

1. Відкрийте свій проєкт на Vercel.
2. Перейдіть у **Settings** (Налаштування).
3. У лівому меню виберіть **«General»**.
4. Прокрутіть до **«Build and Output Settings»**.
5. Натисніть **«Override»** (Перевизначити).
6. Встановіть **Output Directory** = **`dist`** (якщо було `public` — замініть на `dist`).
7. Збережіть зміни.
8. Перейдіть у **Deployments**, відкрийте останній деплой і натисніть **«Redeploy»**.

---

## Коротко

| Що налаштувати   | Значення |
|------------------|----------|
| Root Directory   | `client` |
| Output Directory | `dist`   |
| Змінна           | `VITE_RGS_URL` = `https://ваш-rgs.onrender.com` |

Помилка «не знайдено вихідного каталогу public» зникає, коли в налаштуваннях проєкту вказано **Output Directory: dist**.
